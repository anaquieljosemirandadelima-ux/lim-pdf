# LIM PDF

Portal gratuito de ferramentas PDF com processamento local no navegador, páginas SEO individuais, guias editoriais e preparação para Google Search Console e AdSense.

## Recursos

- 30 ferramentas funcionais.
- 30 páginas estáticas de ferramentas.
- 8 guias originais.
- Sitemap, robots, dados estruturados e canonical.
- Consentimento, políticas legais, acessibilidade e segurança.
- AdSense e Search Console configuráveis por variáveis de ambiente.
- Sem armazenamento dos documentos no servidor.

## Desenvolvimento

```bash
npm ci
npm run dev
```

## Validação

```bash
npm run check
```

## Produção

```bash
npm run build
npm run start
```
